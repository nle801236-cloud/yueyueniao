# Bubble Machine

当前项目已经接成一个可直接运行的 React + Vite 应用，主界面组件位于 [`BubbleMachine.tsx`](/Users/yueyueniao/Documents/Playground/BubbleMachine.tsx)。

## 启动

在当前目录执行：

```bash
npm install
npm run dev
```

然后打开终端里显示的本地地址，通常是 `http://localhost:5173`。

## 构建

```bash
npm run build
```

构建产物会输出到 `dist/` 目录。

## 说明

- `src/main.tsx` 是 React 入口。
- `src/App.tsx` 连接到主组件。
- 旧的 `app.js` 和 `styles.css` 目前保留在目录中，但已经不是新的运行入口。
